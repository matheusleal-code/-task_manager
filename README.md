### Installing

Clone the repo

```
https://github.com/matheusleal-code/-task_manager.git
```

Running backend and open mongo

```
    cd Desktop/-task_manager/api
    npm install
    npm run dev
    mongod
    mongo
```

Running frontend

```
    cd -task_manager
    npm install
    npm start
```

Open localhost on port 3000
